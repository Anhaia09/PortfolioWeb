O que é HTML/HTML5?

HTML (HyperText Markup Language) é a linguagem usada para criar e estruturar páginas web, utilizando tags para definir elementos como textos, imagens, links e tabelas.
HTML5 facilita a criação de páginas web modernas, interativas e acessíveis.


O que é CSS?

Com o CSS, é possível controlar a aparência de elementos, como cores, fontes, espaçamento e layout, separando o conteúdo (HTML) da apresentação.
Isso facilita a manutenção e a consistência visual das páginas web, permitindo aplicar estilos de forma eficiente a múltiplas páginas ao mesmo tempo.


O que é JavaScript?

Diferente de HTML e CSS, que estruturam e estilizam o conteúdo, respectivamente, JavaScript permite criar funcionalidades avançadas, como validação de formulários, animações, manipulação de elementos da página em tempo real. É essencial para o desenvolvimento de aplicações web modernas, tornando as páginas mais responsivas e interativas para os usuários.

Calculadora IMC

Essa atividade combina estruturação de conteúdo (HTML), estilização (CSS) e comportamento dinâmico (JavaScript) para criar uma calculadora funcional de IMC em uma página web.


Calculadora com operações básicas

Esse projeto implementa uma calculadora interativa usando HTML, CSS e JavaScript. A estrutura HTML define a interface da calculadora com botões numéricos e de operações, além de dois displays para mostrar a expressão e o resultado. O CSS estiliza a calculadora, centralizando-a na página e aplicando cores e estilos aos botões e displays. O JavaScript adiciona a funcionalidade da calculadora, permitindo a inserção de números e operações, limpando os displays e calculando o resultado da expressão matemática.


Tela de cadastro

Essa atividade cria uma aplicação de cadastro de Squads utilizando o Modelo de Objeto de Documento (DOM). A estrutura HTML define um formulário para inserir dados de Squad e uma tabela para listar Squads cadastradas. O CSS estiliza a página para garantir uma apresentação organizada e responsiva. O JavaScript adiciona interatividade ao formulário e à tabela: ele captura os dados inseridos no formulário, valida e adiciona novas linhas à tabela quando o formulário é submetido. Além disso, permite editar e excluir Squads existentes na tabela, mantendo uma interface dinâmica e funcional para gerenciar informações de Squads.
